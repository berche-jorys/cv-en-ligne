## 🐍 Assembleur Python (personnel)

*Projet personnel Python visant à simuler un assembleur simple pour une architecture réduite.*

- **Contexte** : travail personnel d’apprentissage sur le langage Python et les principes bas niveau de l’assemblage
- **Technologies** : Python
- **Rôle** : Développeur Python
- **Apprentissages** :
  - compréhension du fonctionnement d’un assembleur (traduction d’instructions en binaire)
  - définition d’une architecture d’instructions simplifiée
  - génération de code machine sous forme de fichier binaire
  - structuration et encodage d’instructions en représentation machine
  - approche des fondamentaux d’un processeur et du langage assembleur

- **Remarque** : le projet ne gère pas l’exécution du code généré ; le binaire produit sert de représentation pédagogique et n’est pas exécutable sur une machine réelle.
