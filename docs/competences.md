## 💻 Langages de programmation

| Langage | Niveau | Utilisation |
|---------|--------|-------------|
| **PHP** | 🟡 Intermédiaire | Développement web backend |
| **C#** | 🟡 Intermédiaire | Programmation orientée objet, Visual Studio, logique métier |
| **HTML/CSS** | 🟡 Intermédiaire | Pages web structurées et stylisées |
| **JavaScript** | 🟡 Débutant | Interactivité frontend |

## 🗄️ Bases de données

- **MySQL** - Requêtes SELECT, INSERT, UPDATE, DELETE
- **Modélisation** - Tables relationnelles
- **Conception** - Tables, clés étrangères, intégrité des données

## 🌐 Développement Web

- **HTML** - Structure sémantique des pages
- **CSS** - Mise en forme, responsive design basique
- **JavaScript** - DOM manipulation, événements

- **Versioning** - Git, GitHub, GitLab
- **Éditeur** - VS Code, Visual Studio
- **Base de données** - PhpMyAdmin

- Déploiement d'applications
